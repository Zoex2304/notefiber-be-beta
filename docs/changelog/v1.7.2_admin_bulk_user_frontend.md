# Frontend Changelog: Admin User Creation & Bulk Upload

## Version: v1.7.1
**Date:** 2026-01-05
**Feature:** Admin User Management Enhancements

## Overview
New features added to the User Management page to allow single user creation and bulk user import via JSON file.

## API Endpoints

### 1. Create Single User
**Endpoint:** `POST /api/admin/users`
**Content-Type:** `application/json`

**Payload:**
```json
{
  "email": "user@example.com",
  "password": "securePassword123",
  "full_name": "New User Name",
  "role": "user" // "user" or "admin"
}
```

### 2. Bulk Create Users
**Endpoint:** `POST /api/admin/users/bulk`
**Content-Type:** `multipart/form-data`

**Request:**
- Form Data Field: `file`
- File Type: `.json`

**File Format (JSON Structure):**
The uploaded JSON file must follow this structure. This can be generated from the lexical editor or uploaded directly.

```json
{
  "users": [
    {
      "email": "alice@company.com",
      "password": "tempPassword001",
      "full_name": "Alice Johnson",
      "role": "user"
    },
    {
      "email": "bob@company.com",
      "password": "tempPassword002",
      "full_name": "Bob Smith",
      "role": "admin"
    }
  ]
}
```

**Response:**
```json
{
  "success": true,
  "data": {
    "created_count": 1,
    "failed_count": 1,
    "results": [
      {
        "email": "alice@company.com",
        "success": true,
        "id": "uuid-..."
      },
      {
        "email": "bob@company.com",
        "success": false,
        "error": "User already exists"
      }
    ]
  }
}
```
